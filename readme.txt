== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'ChatMe Cookie Consent'
3. Activate ChatMe Cookie Consent from your Plugins page.
4. Click Options from your Plugin page

= From WordPress.org =

1. Download ChatMe Cookie Consent.
2. Upload the 'jquery-cookie-law' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate ChatMe Cookie Consent from your Plugins page.
4. Click Options from your Plugin page

= Extra =

1. Adjust the CSS of your theme as needed, to make everything pretty.
